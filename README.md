# BigBrotr Presentation

Interactive slides for the BigBrotr project - an open-source measurement infrastructure for the Nostr decentralized network.

## Quick Start

```bash
npm install
npm start
```

Opens at `http://localhost:3000`

## Navigation

- `→` / `Space`: Next slide
- `←`: Previous slide
- Click the file icon (top right): Toggle speaker notes

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
